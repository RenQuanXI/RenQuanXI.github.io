# 3.基本数据类型和引用数据类型

  1. 基本数据类型：简单的数据类型也叫值类型，有数字Number，字符串String，布尔值Boolean，未定义Undefind，空Null

    引用数据类型：复杂的数据类型也叫引用类型，有数组 ==Array==，函数==Functiom==，对象，等等。

  2. 内存中的栈和堆

    栈区存放 基本数据类型 和 引用数据类型的地址

    堆区存放 引用类型数据的数据

  3. new关键字的执行过程

    • new关键字先向内存空间中申请一片区域存放将要创建的对象

    • 在构建函数内部，this关键字会指向当前申请的内存空间

    • 然后构建函数会 通过this向内存空间中存放属性和方法

    • 最后将创建好的对象（已经拥有属性和方法）返回给外部的变量名

  4. 基本数据类型在复制时（一个变量名赋值给另一变量名），会在栈区申请一块空间安存放一个新的数据，并且变量名会指向新的数据

  5. ​ 引用类型数据在复制时（一个变量名赋值给另一个新的变量名），会在栈去申请一块的空间存放堆区中的引用类型数据的地址，并且新的变量名会指向栈区中存放地址的空间
